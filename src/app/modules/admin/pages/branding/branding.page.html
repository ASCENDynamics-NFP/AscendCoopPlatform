Branding Editor Preview

How branding works

Branding values flow from defaults → Firebase Remote Config → per-device override (highest wins). Use this page to preview a brand on this device, then copy the generated JSON and upload it to the Firebase Remote Config console to apply it for everyone.

The AGPL footer and license attribution are not configurable — the project license requires them to remain visible.

Publish

When you're happy with the draft, generate the Remote Config JSON and upload it via the Firebase console:

  1. Click Copy Remote Config JSON below.
  2. Open the Remote Config console and pick your project.
  3. Use Publish from JSON (or paste each parameter individually) and confirm publish.

firebase deploy --only remoteconfig is not used in this project — uploads happen through the Firebase console UI.
