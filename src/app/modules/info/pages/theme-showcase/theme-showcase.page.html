What is the design system?
A set of design tokens and global primitives that every page consumes for consistent theming.
How do I theme my page?
Use .app-section, .app-section-header, and --app-* CSS variables instead of hard-coded values.
Does dark mode work automatically?
Yes — both the OS preference and the manual toggle swap the --app-* tokens for you.
